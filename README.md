## CRUD operations , leaf nodes are files in the root of this repo.

```mermaid
graph TD;
    id1[(CRUD)]-->id2(Create);
    id1-->id3(Read);
    id1-->id4(Update);
    id1-->id5(Delete);
    id3-->id6([GET-read data]);
    id2-->id7([POST - create data]);
    id5-->id8([POST - delete data]);
    id4-->id9([POST - update data]);
```
